Reversi
=======

###A small django app to play reversi with a friend or against an AI

Version 1.0 - 2014_04_23

How to install:
-------
 - download the project
 - run the following : 
 ```python
 python manage.py syncdb```

How to play:
------- 
 - run the following : 
 ```python
 python manage.py runserver```
 - go to http://127.0.0.1:8000/new/
 - play
